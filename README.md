# Rep AI — Public Changelog

This repo is the public-facing changelog for **Rep AI**, an AI-powered push-up counter for iOS and Android.

The source code is private. This repo exists so anyone can follow along with what's being built and changed.

---

## What is Rep AI?

Rep AI uses your phone's camera and on-device AI to count push-up reps in real time — no wearable, no subscription.

- Counts reps automatically using pose detection
- Tracks good vs bad form per rep
- Works with standard, wide, diamond, knee, pike, and decline push-ups
- Custom sets with rest timers
- Streak tracking, goals, and workout history
